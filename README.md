# Sumyyah Saeed — Personal Portfolio Website

A responsive personal portfolio built with pure **HTML5** and **CSS3** — no frameworks, no libraries. Designed with a **FinTech Midnight** aesthetic theme to reflect my background in blockchain development, financial technologies, and data analytics.

---

## 🛠️ Built With

- **HTML5** — semantic structure and forms
- **CSS3** — Flexbox, Grid, animations, media queries, custom properties
- **Google Fonts** — Syne (headings) + DM Sans (body)
- No Bootstrap. No Tailwind. No CSS frameworks.

---

## ✨ Features

- Fully responsive — mobile, tablet, and desktop breakpoints
- CSS Grid and Flexbox layouts throughout
- CSS `@keyframes` animations with stagger delays
- Animated skill progress bars
- Sticky navigation with hamburger menu for mobile
- Working contact form with JavaScript validation
- Back-to-top button
- Consistent shared `style.css` with CSS custom properties
